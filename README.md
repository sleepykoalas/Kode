# Kode!

Kode! is a simple and lightweight text-based editor designed to provide an efficient and minimalistic workflow. It prioritizes security and user privacy through hashed credential storage.

## Features

- Minimal and fast performance  
- Secure user signup with hashed usernames and passwords  
- Clear screen functionality optimized for Linux and macOS environments  
- Text-based interface without graphical distractions  
- Focus on privacy and security  

## Installation

### Prerequisites

- Python 3.x installed on your system.  

#### How to install Python

- **Linux/macOS:** Python 3 is usually pre-installed. To check, run:  
  ```bash
  python3 --version

If missing, install via your package manager, e.g.:

sudo apt-get install python3   # Debian/Ubuntu  
brew install python            # macOS with Homebrew

    Windows:

        Download the latest Python 3 installer from the official site:
        https://www.python.org/downloads/windows/

        Run the installer and make sure to check "Add Python to PATH" during setup.

        Verify installation in Command Prompt:

        python --version

Clone and Run

    Clone the repository from GitHub (works on Linux, macOS, and Windows):

git clone https://github.com/sleepykoalas/kode.git

Navigate to the project directory:

cd kode

Run the script:

    On Linux/macOS:

python3 kode.py

On Windows (Command Prompt or PowerShell):

        python kode.py

Usage

    When you run the script, you will be prompted to create a username and a secure password.

    Your credentials will be securely hashed before storage.

    Use the text-based interface to manage notes and tasks efficiently.
