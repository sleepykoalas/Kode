# Kode!

Kode! is a simple, lightweight, text-based editor designed to provide an efficient and minimalistic workflow. It prioritizes user privacy and security.

## Features

- Minimal and fast performance  
- Secure user signup with recovery code  
- Clear screen functionality optimized for multiple OS environments  
- Text-based interface without graphical distractions  
- Focus on privacy and simplicity  

## Installation

Requires a C compiler (e.g., gcc).

Compile the program with:

```bash
gcc -o kode kode.c

Run the program:

./kode

Usage

Follow the on-screen prompts to create a username and secure password. You will receive a recovery code that will only be shown once. Type "I'm Done" exactly to proceed.

Enjoy coding with Kode! Keep it simple, keep it secure.
